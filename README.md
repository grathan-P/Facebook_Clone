## Features
- Mobile-first responsive layout
- Two-column layout on desktop, stacked layout on mobile
- Responsive footer using `flex` and `flex-wrap`
- Clean alignment without negative margins
- Pixel-consistent spacing and typography
- Static UI clone (no backend or authentication)

---

## Responsiveness
The layout adapts across breakpoints:
- **Mobile:** Single-column, centered content
- **Tablet:** Improved spacing and readability
- **Desktop:** Two-column layout with constrained width

Footer links wrap naturally instead of shrinking, ensuring readability on all screen sizes.

---

## What I Learned
- Mobile-first layout design using Tailwind CSS
- Difference between `flex-wrap` and CSS Grid for content-heavy footers
- Proper use of containers instead of margin-based alignment
- Importance of `min-h-screen` for vertical centering
- Writing cleaner, more maintainable utility-first CSS
